🐳 Docker Deep Dive

A hands-on Docker course that starts from the terminal up: first the Linux floor every container stands on, then Docker itself. Lessons are published here as the course progresses — check back, or watch the repo.

labs repo: pdfarhad/docker-hands-on · reference: Linux & shell cheatsheet

How this course works

  • Read a lesson → run its lab chapter → take the self-check quiz. Labs live in the labs repo; clone once, git pull before each lesson.
  • Everything runs on macOS, Windows, or Linux — labs happen inside a disposable Ubuntu container, identical on every OS.
  • The ☰ button (top right, on every page) jumps between lessons; diagrams marked ▶ play are animated — step through them.
  • Quizzes grade themselves in your browser — commit to an answer before revealing.
